1. Hover Interactions

Event API

socilab-animations bind deeply natively to user interaction. Hover over this button to natively trigger a complex staggered Jelly effect that automatically reverses perfectly upon exit without dropping frames!

app.ts 
import { Jelly } from 'socilab-animations';

// Bind to Hover state with native Reverse
new Jelly('#hover-demo-btn', { 
    trigger: 'hover', 
    reverseOnLeave: true,
    stagger: 30, // ms between each char
    duration: 600,
    easing: 'ease-out'
});
// Automatically tracks WAAPI PlaybackRate!
router.ts 
import { CameraFlyThrough } from 'socilab-animations';

const mainPage = document.getElementById('main-content');
const nextRoute = document.getElementById('page-layer');

button.addEventListener('click', async () => {
    nextRoute.style.display = 'flex';

    // Cinematic 3D Camera Fly-through
    // outgoing scales massively past the camera
    // incoming slides in focus from depth
    await new CameraFlyThrough(
        mainPage, 
        nextRoute, 
        { duration: 1200 }
    );
});

2. Page Transitions

Core Engine

The generic `TransitionEngine` guarantees perfectly synchronized outgoing and incoming layout layers. It natively computes promises internally only when both animations hit their offset 1.0!

Scroll Interactivity Native API

The library utilizes highly optimized Intersection Observers to trigger and infinitely loop/reverse animations on scroll threshold.
